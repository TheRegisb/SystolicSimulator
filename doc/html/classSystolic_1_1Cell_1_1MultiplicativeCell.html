Detailed Description

Implementation of an ICell for multiplicative operation.

Cell that performs X*Y computation, where Y is a constant integer defined at cell creation.

◆ compute()

std::tuple< std::optional< int >, std::optional< int > > Systolic::Cell::MultiplicativeCell::compute ( )
overridevirtual

Perform the computation.

Does an X*Y computation where X is its internal stored value and Y is the constant defined at the cell creation. Replaces its internal stored value by the computed result.

Returns
A tuple with at 0 the new computed value and at 1 the initial value from the input queue. May be empty on empty feeding.

Implements Systolic::Cell::ICell.
