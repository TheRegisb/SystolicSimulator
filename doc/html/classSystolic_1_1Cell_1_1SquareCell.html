Detailed Description

Implementation of an ICell for square operation.

Cell that performs X^2 computation.

std::tuple< int, std::optional< int > > Systolic::Cell::SquareCell::compute ( )
overridevirtual

Perform the computation.

Does an power-of-two computation based on its internal stored value. Replaces its internal stored value by the computed result.

Returns
A tuple with at 0 the sum of all values that passed down that cell and at 1 the computed value (may be empty on empty value computation).
